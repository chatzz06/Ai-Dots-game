# Ai-Dots-game

An interactive AI-powered Dots Game built using HTML, CSS, and JavaScript. Challenge the computer in a classic dots-and-boxes game where strategy and planning determine the winner.

📂 Repository

https://github.com/chatzz06/Ai-Dots-game


✨ Features

🤖 Play against an AI opponent
🎯 Interactive and responsive game board
📊 Real-time score tracking
🔄 Restart game functionality
🎨 Clean and modern user interface
⚡ Smooth gameplay using JavaScript

## ▶️ How to Run
1. Open project folder 
2. Install **Live Server** extension
3. Open `index.html`
4. Click **“Open with Live Server”**
5. Play the game in browser 🎮

## 🎮 How to Play

- Click on a line between two dots to claim it.
- Complete a box to earn a point.
- Completing a box gives you another turn.
- The AI automatically makes its move after yours.
- The player with the most boxes at the end wins.
